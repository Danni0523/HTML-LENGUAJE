Lenguaje HTML

¿Qué es HTML?

El lenguaje de marcado de hipertexto (HTML) es el código utilizado para estructurar y mostrar páginas web y su contenido. Los ejemplos de contenido incluyen párrafos, viñetas, imágenes, tablas de datos, etc.

Características principales del Lenguaje HTMOL:

Una de las características de HTML es la forma en que se divide la información. Esto se realiza mediante dos bloques principales: el HTML o elemento de cabecera y el cuerpo

    |Meta Datos

Es la información de la cual se le agrega a una pagina web y se le conoce como contexto .

    |Referencia a ficheros

Son los archivos JavaScript y CSS. los cuales son los encargados de darle estilo a la pagina web

    |Scripts y estilos.

Los estilos se pueden dejar afuera de los ficheros y esta pagina la reconoce como estilo debido a sus etiquetas.
