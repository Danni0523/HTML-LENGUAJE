Elementos Basicos utilizados en HTML

Los elemetos basicos son los siguientes

Somos una empresa de impresión a 3D que desde el 2021 se dedica a realizar figuras y pedidos a medidas en material PLA contamos tienda virtual y envíos a todo el país.

    |1.body

Se usa para el contenido ya que se evidencia la apertura y cierre del contenido de una pagina web

    |2.Head

En la parte superior del documento HTML, puede especificar metadatos como el título del documento, la hoja de estilo, JavaScript y CSS.

    |3.div

Elemento que se utiliza principalmente para agrupar otros elementos y actuar como plantilla para otros controles. Las etiquetas div a estructurar su documento en secciones..

    |4.a

Esta es una etiqueta útil para vincular a páginas web. El atributo principal de la etiqueta HTML es el href, donde pones el enlace al que te quieres conectar..

    |5.strong

Si tiene mucho texto, es importante poder resaltar ciertas partes. Con etiquetas poderosas puedes hacer esto.

    |6.br

para decirle al navegador que se avecina un salto de línea. Ayuda a que el texto sea más fácil de leer..

    |7.H1-H6

Nos permiten mostrar la importancia de los títulos y la estructura del contenido, de esta forma ayudamos a los bots a entender la importancia del contenido. .

    |8.IMG

para mostrar imágenes en el contenido. Necesita el atributo src para funcionar, porque ahí es donde le indicaremos dónde debe mostrar la imagen..

    |9.ol, li, / ul, li, / li, ul

Las etiquetas OL y LI se utilizan para crear listas, OL para listas ordenadas y UL para listas no ordenadas. Dentro de la lista, los elementos están marcados con la etiqueta LI..

    |10.p

Las etiquetas se utilizan para agrupar texto en un párrafo. El objetivo es poder hacer que el contenido sea más fácil de leer y más organizado..

    |11.SPAN

permiten personalizar el estilo de solo una parte del texto..
